## #-english-version 
### Features
- User-friendly interface and experience 😌
- Batch editing features 📦
- Detailed statistics 📊
- Export assistant 🗄️

### Installation
1. Clone the repository: `git clone https://github.com/NyxAwroo/Datasets-Images-EditSelect.git`
2. Navigate to directory: `cd Datasets-Images-EditSelect`
3. Install dependencies: `npm install`

### Recommended Workflow
```
1. Import your images. 
2. Choose desired editing features. 
3. Apply changes. 
4. Export results.
```

### Project Structure
- `/src`: Source files
- `/assets`: Images and other assets

### Use Cases
- Personal projects 
- Commercial applications
