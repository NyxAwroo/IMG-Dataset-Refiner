# Datasets Images Edit Select

## English Documentation
This repository contains various datasets and tools for image editing and selection. It aims to facilitate research and development in the field of image processing.

### Usage
To use the datasets, simply clone the repository and access the relevant files. Each dataset is stored in its own directory.
